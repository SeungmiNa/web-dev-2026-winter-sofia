Understanding Image Optimization and Accessibility

This week, I learned that choosing the right image format is essential for web performance. The textbook explains that photographs work best as JPEG files because they support millions of colors, while logos or simple graphics with flat colors should be saved as GIF or PNG files because they support transparency and lossless compression. I was surprised to learn that we should avoid using spaces or punctuation in file names and always use lowercase letters to ensure the links work correctly on all servers.

I also discovered that the number one reason visitors leave a website is because the page takes too long to download. Therefore, we must optimize images to reduce file size without losing quality. Furthermore, accessibility is critical. We must always include the alt attribute in our image tags. This provides a text description for visitors who use screen readers, ensuring that our website is inclusive for everyone, including those with vision challenges.
